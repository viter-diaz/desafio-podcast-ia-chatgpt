# Prompt para criação do roteiro

Crie um roteiro curto para um podcast chamado "IA em Minutos".

Tema do episódio:

5 dicas para usar o ChatGPT nos estudos.

Regras:

- duração aproximada de 3 minutos;
- linguagem simples e descontraída;
- começar com uma apresentação curta;
- apresentar cinco dicas práticas;
- não utilizar termos técnicos difíceis;
- finalizar convidando o ouvinte a continuar estudando;
- escrever apenas o texto que será narrado.
